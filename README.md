# RangeScopes

Gem adds scopes to active record model that is related with ranges.

Scopes:
  * range_between("range_from" => 15, "range_till" => 20)

## Installation

Add this line to your application's Gemfile:

    gem 'range_scopes'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install range_scopes
